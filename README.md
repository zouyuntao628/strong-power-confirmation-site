# 1楼强电设计确认网站

这是一个静态确认网站，用于展示并确认《1楼 -调整设备间距.3dm》的强电设计、设备识别、工程量和报价估算。

## 内容

- `index.html`：网页主体
- `styles.css`：页面样式
- `script.js`：确认状态、报价切换、导出摘要交互
- `assets/plan.svg`：由 3DM 图层抽取生成的简化平面示意图

## 使用

直接打开 `index.html` 即可查看。确认状态保存在当前浏览器的 `localStorage` 中。

## 发布到 GitHub Pages

仓库推送到 GitHub 后，可在仓库 Settings -> Pages 中选择 `main` 分支根目录发布。
